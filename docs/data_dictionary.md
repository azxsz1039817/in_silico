# Data Dictionary

## Prompt Metadata

`data/raw/prompts/covid_prompt_metadata.csv`

- Canonical COVID prompt metadata used for analysis and consistency checks.
- Important columns: `issue_no`, `prompt`, `keywords`, `endings`, `labels`, `topic_no`, `reverse_code`.

`data/raw/prompts/covid_generation_prompt_bank.csv`

- Generation-era COVID prompt bank retained for archival data-generation provenance.

`data/raw/prompts/validation_prompt_metadata.csv`

- Prompt metadata for the known-issue validation analysis.

`data/raw/prompts/projection_labels.csv`

- Semantic-axis anchor labels used in the embedding projection workflow.

## Cached Response/Projection Inputs

`data/raw/responses/covid_response_projections.csv.gz`

- Compressed canonical input for the COVID analyses.
- Derived from the historical `output_wave2/completions_proj_*.txt` files.
- Unpacked locally to `data/raw/responses/covid_response_projections.csv` by `scripts/unpack_raw_data.py`.

`data/raw/responses/validation_response_projections.csv.gz`

- Compressed canonical input for the validation analyses.
- Derived from the historical `output_validation_wave2/completions_proj_*.txt` files.
- Unpacked locally to `data/raw/responses/validation_response_projections.csv` by `scripts/unpack_raw_data.py`.

Important columns:

- `completion`: generated GPT-3 continuation.
- `completion_no`: completion index within prompt.
- `prompt_no`: prompt identifier.
- `question_no`: question wording identifier.
- `issue_no`: issue identifier.
- `issue_no_key`: issue identifier plus keyword variant.
- `keyword`: policy phrase variant.
- `ending`: prompt ending text.
- `end_type`: normalized ending label.
- `labels`: semantic-axis anchor labels.
- `partisanship`: prompt-prime code.
- `liberal`: `1` for liberal prime, `0` for conservative prime.
- `named_prime`: `1` for named partisan prompts used in manuscript coefficient plots.
- `proj`: semantic-axis projection score.
- `proj_b`: binary sign of `proj`.
- `full_prompt`: full prompt text given to GPT-3.

## Justification Metadata

`data/raw/justifications/justification_cache_summary.csv`

- Compact summary of the archived justification text/embedding cache.
- Inventories the archived embedding pickle files used in the original exploratory clustering workflow.

`data/raw/justifications/justification_texts.csv.gz`

- Compressed full-text justification data reconstructed from the archived `pos_lst_*` and `neg_lst_*` pickle files.
- Unpacked locally to `data/raw/justifications/justification_texts.csv` by `scripts/unpack_raw_data.py`.
- Important columns: `prompt_no`, `connector`, `sentiment`, `justification_no`, `justification`, `source_file`.

`data/raw/justifications/justification_embeddings_float32_part*.npz`

- Four float32 NumPy archives of the archived justification embeddings.
- Each archive contains one dense array named `embeddings`; together they contain 66,000 rows and 1,536 columns.
- The files are split so each remains below GitHub's 100 MB per-file limit.
- Replaces the much larger original embedding pickle cache.

`data/raw/justifications/justification_embeddings_manifest.csv`

- Slice manifest for `justification_embeddings_float32_part*.npz`.
- Important columns: `prompt_no`, `connector`, `sentiment`, `archive_file`, `archive_offset`, `global_offset`, `n`, `dimensions`, `source_embedding_file`, `source_text_file`.

## Generated Outputs

`data/processed/`

- Normalized prompt tables and assembled response datasets generated by `scripts/run_all.py`.

`tables/`

- Coefficient tables, forecast binomial tests, justification proportion tests, justification bar-chart counts, and consistency checks.

`figures/`

- Regenerated analytic figures. COVID topic coefficient figures are faceted coefficient panels; validation coefficient figures are split into two faceted panels; justification figures are stacked liberal/conservative count bars by cluster/theme. Mixed-axis coefficient figures label each facet with its own semantic-axis contrast.
